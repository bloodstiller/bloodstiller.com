Linux Targets:

## If Access via Domain Account

### Active Directory Integration

  • Kerberos Tickets

    • Dump tickets: mimikatz, klist
    • Golden ticket creation
    • Pass-the-Ticket attacks

  • Samba/Winbind Configuration

    • Location: /etc/samba/smb.conf
    • Check for sensitive configurations, stored credentials

### Network Shares

  • NFS Shares

    • Location: /etc/exports
    • Look for misconfigured or sensitive NFS exports

  • Samba Shares

    • Location: /etc/samba/smb.conf
    • Check for shared directories, accessible files

### Local and Domain Policies

  • Password Policies

    • Location: /etc/login.defs, /etc/security/pwquality.conf
    • Minimum password length, complexity requirements
    • Password expiration settings

  • User Rights Assignment

    • Location: /etc/sudoers, /etc/sudoers.d/*
    • Check for sudo privileges, special permissions

### Sensitive Files and Directories

  • User Profiles

    • Location: /home/%username%
    • Target Documents, Desktop, Downloads, .ssh directory

  • Backup Files

    • Common locations: /var/backups, /mnt/backups

    • Look for rsnapshot, tar, or other backup files

### Installed Software and Services

  • Credential Stores

    • Password Managers (e.g., KeePass databases)
    • SSH keys: home/$USERNAME.ssh
    • UNSHADOW! Shadow/ etc

  • Database Servers

    • MySQL: /etc/mysql/my.cnf
    • PostgreSQL: /etc/postgresql/*/main/pg_hba.conf
    • Check for configuration files with stored credentials

  • Email Servers

    • Postfix, Dovecot: /etc/postfix/main.cf, /etc/dovecot/dovecot.conf
    • Check for configuration files with stored credentials

### Miscellaneous Targets

  • Scheduled Tasks and Cron Jobs

    • Location: /etc/crontab, /etc/cron.*, /var/spool/cron/crontabs
    • Look for sensitive scripts, backup jobs

  • Startup Items

    • Location: /etc/init.d, /etc/systemd/system
    • Check for sensitive services, startup scripts

## If Access via Local Account

### Registry and Configuration Files

  • Shadow File

    • Location: /etc/shadow
    • Contains hashed user passwords

  • Passwd File

    • Location: /etc/passwd
    • Contains user account information

  • Group File

    • Location: /etc/group
    • Contains group information

### Sensitive Files and Directories

  • User Profiles

  • Credential Files

    • SSH keys: home/%username%.ssh
    • GPG keys: home/%username%.gnupg
